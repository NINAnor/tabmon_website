📊 Live Monitoring Dashboard

Explore TABMON’s real-time monitoring dashboard — your comprehensive portal to Europe’s acoustic biodiversity network. Our interactive platform provides live insights into data acquisition, device performance, and biodiversity patterns across all monitoring sites.

  • 100+ active sites monitored in near real-time
  • 25+ terabytes of high-quality acoustic recordings acquired and processed
  • <24-hour latency from field recording to dashboard visualization and analysis
📈 Access the TABMON Dashboard — Real-time data from across Europe

The dashboard features:

  • Device status & network coverage — Monitor which recorders are operational and their geographic distribution
  • Data ingestion metrics — Track upload frequency, data quality, and processing status
  • Preliminary biodiversity patterns — Visualize species occurrence, activity levels, and temporal trends
  • Quality metrics — Device health, audio signal diagnostics, and data completeness statistics

📝 Data Annotation Platform

To enhance our AI models and improve biodiversity monitoring accuracy, we invite citizen scientists and acoustic ecologists to contribute to our TABMON Listening Lab data annotation platform. By labeling acoustic samples, you help us:

  • Train more accurate species recognition algorithms
  • Improve confidence in automated detections
  • Contribute to open science in bioacoustics
🎧 Access the TABMON Listening Lab — Help us annotate acoustic data!

💻 Technical Resources & Code Repositories

TABMON is committed to reproducible, open science. All code, configurations, and model documentation are made publicly available under appropriate open-source licenses.

🏠 Main Project Hub

  • Repository: https://github.com/NINAnor/TABMON
  • Purpose: Central documentation hub linking all TABMON project repositories, protocols, and metadata standards
  • Contents: Project overview, work package descriptions, partner information, and links to all technical resources

🤖 AI Classification & Analysis Engine

  • Repository: https://github.com/BenMcEwen1/TABMON-Classification-Pipeline
  • Purpose: End-to-end AI pipeline from raw field audio to inference of Essential Biodiversity Variables (EBVs)
  • Contents:
    • Species detection and classification models
    • Acoustic feature extraction workflows
    • Uncertainty quantification and propagation
    • Community-level metric computation
    • EBV aggregation and reporting

📊 TABMON Report Generator

  • Repository: https://github.com/NINAnor/tabmon_reporting
  • Purpose: Automated generation of TABMON yearly and seasonal monitoring reports
  • Features:
    • Site-level and cluster-level summary statistics
    • Temporal trend analysis
    • Species occurrence and activity patterns
    • Publication-ready visualizations and tables
    • Integration with Zenodo for open archival

📋 Data & Reports on Zenodo

🔗 Data Access & Integration

TABMON data are being integrated into major biodiversity data portals following FAIR principles:

  • GBIF (Global Biodiversity Information Facility) — Species occurrence data
  • EBV Data Portal — Essential Biodiversity Variable indicators
  • National monitoring schemes — Integration with country-specific biodiversity databases
  • Open science repositories — Zenodo, OSF, and institutional repositories

Data Citation & Licensing

All TABMON datasets include comprehensive metadata, versioning, and clear licensing information. We follow open data best practices with DOI registration through Zenodo and GBIF.
